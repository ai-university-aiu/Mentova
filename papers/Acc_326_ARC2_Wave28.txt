The rule identifies a vertical 5-divider column, slides all 0-cell
4-connected components to the left of the divider rightward until each
touches column DC-1 (one column left of the divider), then fills all cells
right of the divider with value 2 in every row that the slide left open.

TASK DESCRIPTION

Task 6e453dd6 presents grids containing a vertical column of 5s (the
divider at column DC) and one or more 0-cell objects on the left side.
Each object is a 4-connected component. The transformation slides every
object rightward so its rightmost cell abuts the divider. After sliding,
rows where a gap (0 at DC-1) is adjacent to a non-zero cell (at DC-2) and
another 0-cell object still exists further left (in columns 0..DC-3) are
"open." Open rows are filled right of the divider with value 2.

RULE: SLIDE-OPEN

Step 1 - Find the divider column.

Locate column DC where every cell in the column holds value 5. This is
found by trying candidates with between/3 and verifying all rows with
forall.

Step 2 - Partition 0-cells into components.

Collect all (R, C) pairs where Grid[R][C] = 0 and C < DC. Partition them
into 4-connected components using iterative DFS flood-fill.

Step 3 - Slide each component.

For each component, compute Shift = (DC - 1) - MaxCol, where MaxCol is
the rightmost column in the component. In pass 1, erase all component
cells (set to placeholder 6). In pass 2, place each cell at column C +
Shift. The two-pass erase-then-place design prevents self-overwrite when
shifted positions overlap original positions within the same component.

Step 4 - Fill open rows.

For each row R after all slides:
- VRight = cell(R, DC-1). If VRight = 0, there is a gap at the divider.
- VGap = cell(R, DC-2). If VGap != 0, the gap is isolated.
- If there exists at least one 0 in columns 0..DC-3, another object is
  still present to the left.
If all three conditions hold, fill columns DC+1 through NCols-1 with 2.

IMPLEMENTATION

The implementation uses these key predicates:

arc2_so_divider_(Grid, NRows, DC)
    Finds the divider column. Generates candidates with between/3,
    builds a row list with numlist/3 (after computing NRowsM with is/2),
    and checks every row with forall.

arc2_so_comps4_(Cells, Comps) / arc2_so_comps4_iter_/4 / arc2_so_flood4_/6
    Standard 4-connected component flood-fill over the list of 0-cells.

arc2_so_slide_(DC, _NRows, _NCols, Comp, G0, G1)
    Two-foldl erase-then-place slide for one component. Placeholder value
    6 is never present in the output because it is always overwritten.

arc2_so_fill_open_(Grid, NRows, NCols, DC, Out)
    Iterates over all rows using foldl. For each row that satisfies all
    three open-row conditions, fills columns DC+1..NCols-1 with 2.
