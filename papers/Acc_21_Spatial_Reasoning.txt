ABSTRACT

We report Rung 21 of Mentova's 48-rung reasoning ladder: spatial reasoning.
Mentova resolves containment and position using reference frames. The cat is on
the mat; the mat is in the kitchen; the kitchen is in the house. Mentova returns
the full containment chain [cat, mat, kitchen, house] and confirms that the cat
is in the house transitively. Pass criterion met: transitive location correct.


INTRODUCTION

Spatial reasoning requires resolving where entities are located relative to
reference frames. The classic challenge is transitive containment: if the cat
is on the mat and the mat is in the kitchen, where is the cat? Rung 21 equips
Mentova with a spatial containment engine that traverses nested reference frames.


BACKGROUND

Spatial reasoning has been studied in AI since early work on scene understanding.
The Region Connection Calculus (Randell et al., 1992) and other qualitative spatial
formalisms provide algebras over spatial relations. Mentova's implementation uses
a simpler but transparent in/on containment hierarchy with transitive closure.


METHOD

A new module, src/mentova/spatial.pl, implements:

    mentova_spatial(+Query, -Result, -Justification)

Facts: in(X,Y), on(X,Y), near(X,Y) — direct spatial relations.

Supported queries:
    chain(X)           full containment chain from X to root
    in_trans(X, Y)     is X in Y transitively? (with chain)
    where_is(X)        ultimate container of X
    on_top_of(Y)       what is on Y?
    near(X, Y)         proximity check


DEMONSTRATION

    % Query Mentova for spatial reasoning about 'chain(cat)'.
    ?- mentova_query(spatial, chain(cat), R).
    R = answer(chain(cat, [cat, mat, kitchen, house]), ...)

    cat on mat in kitchen in house. Full chain returned.

    % Query Mentova for spatial reasoning about 'in_trans(cat'.
    ?- mentova_query(spatial, in_trans(cat, house), R).
    R = answer(yes, just(transitive_in(cat, house, yes, chain([cat, mat, kitchen, house]))))

    Cat is in house (transitively). Chain shown.


RESULTS

Both test queries return correct spatial answers with full chain justification.
Pass criterion met: transitive location correct (cat on mat in kitchen).


SIGNIFICANCE

Spatial reasoning is essential for robot navigation, scene understanding, and
natural language queries about location. The glass-box chain justification shows
every containment step.


HONEST LIMITATIONS

The containment model is strictly hierarchical (tree structure). It does not handle
partial overlap, regions, or metric distances. The reference frames are static.


REPRODUCIBILITY

Requirements: SWI-Prolog 9.0.4 or later.

  git clone https://github.com/ai-university-aiu/Mentova.git
  cd Mentova
  % Load the Mentova bootstrap into SWI-Prolog, execute the query, and halt.
  swipl -l src/mentova/mentova.pl \
        % Supply a goal or termination instruction to the SWI-Prolog command.
        -g "mentova_boot, \
            % Query Mentova for spatial reasoning about 'chain(cat)'.
            mentova_query(spatial, chain(cat), R1), write(R1), nl, \
            % Query Mentova for spatial reasoning about 'in_trans(cat'.
            mentova_query(spatial, in_trans(cat,house), R2), write(R2), nl" \
        % Supply a goal or termination instruction to the SWI-Prolog command.
        -t halt
