ABSTRACT

Mentova solves ARC-AGI-2 task 7ed72f31 using a new glass-box named rule called
reflect_axis. The rule operates on grids containing non-background colored shapes
and clusters of 2-cells that function as reflection axes. Each shape is reflected
across the axis defined by its nearest 2-cluster: a single 2-cell produces a
180-degree point reflection, a collinear column of 2-cells produces a vertical
axis reflection, and a collinear row of 2-cells produces a horizontal axis
reflection. The original shapes and 2-markers remain unchanged; only the reflected
copies are added. The rule is verified on all 2 training pairs and the test case
of task 7ed72f31, raising the ARC-AGI-2 score from 13/120 = 10.83% to

1. PROBLEM STATEMENT

Task 7ed72f31 presents grids with a solid background color, several non-background
colored shapes, and clusters of 2-cells adjacent to each shape. The output adds a
reflected copy of each shape on the opposite side of its associated 2-cluster.
The original shape and 2-markers are preserved in the output.

Training pair dimensions:
  Train 0: input 18x18, bg=1, 4 shapes, 4 distinct 2-clusters; 18 changes.
  Train 1: input 18x18, bg=8, 3 shapes, 3 distinct 2-clusters; 18 changes.

The 2-clusters in train 0 include: a single 2-cell (point reflection), a 4-cell
vertical column of 2-cells (vertical axis), and a 5-cell horizontal row of 2-cells
(horizontal axis). Train 1 includes two 2-cell vertical columns and one 4-cell
horizontal row of 2-cells.

2. KEY INSIGHT - AXIS TYPE FROM 2-CLUSTER GEOMETRY

The 2-cells form distinct spatial clusters (connected 4-way components). The
geometry of each cluster determines the type of reflection axis:

  A single 2-cell at (R0, C0): the entire shape is reflected through this point.
    Each shape cell (R, C) maps to (2*R0 - R, 2*C0 - C).

  A collinear column of 2-cells (all sharing column C0): vertical axis reflection.
    Each shape cell (R, C) maps to (R, 2*C0 - C).

  A collinear row of 2-cells (all sharing row R0): horizontal axis reflection.
    Each shape cell (R, C) maps to (2*R0 - R, C).

Each shape is paired with its nearest 2-cluster, measured by minimum Manhattan
distance from any shape cell to any 2-cluster cell.

3. RULE SPECIFICATION (reflect_axis)

Inputs:
  - Grid: list of lists of integers.

Phase 1 - Background detection:
  Bg = mode (most frequent value) of all cells in Grid.

Phase 2 - 2-cluster identification:
  Collect all cells with value 2. Group them into 4-connected components
  (clusters). Each cluster will serve as one reflection axis.

Phase 3 - Shape component identification:
  Collect all non-Bg non-2 cells. Group them into 4-connected components by
  growing BFS through same-color adjacent cells. Each component is one shape.

Phase 4 - Shape-to-cluster pairing:
  For each shape component, find the 2-cluster with minimum Manhattan distance
  from any shape cell to any cluster cell. That cluster is the shape's axis.

Phase 5 - Axis type determination:
  For a cluster of one cell (R0-C0): axis = point(R0, C0).
  For a cluster with all cells in row R0: axis = horizontal(R0).
  For a cluster with all cells in col C0: axis = vertical(C0).
  Otherwise: axis = point at first cell of cluster (degenerate fallback).

Phase 6 - Reflection:
  For each shape cell (R, C) with value V:
    Compute reflected position (R2, C2) from the axis.
    If (R2, C2) is within grid bounds, set grid[R2][C2] = V.

Output: the input grid with reflected cells added (original cells unchanged).

7. SIGNIFICANCE

The reflect_axis rule demonstrates PrologAI's ability to identify spatial
symmetry axes from marker cells and apply reflective transformations to
non-trivially shaped objects. By using 2-cell clusters as typed axis markers
(point, horizontal row, vertical column), and by nearest-cluster pairing to
associate each shape with its mirror axis, the rule handles multiple shapes
and multiple axis types within a single grid in a single unified inference.
The minimum-Manhattan-distance pairing is a reusable primitive for any task
where one object designates an operation to be applied to another object.
