Mentova just solved its sixth ARC-AGI-2 evaluation task using a new glass-box rule called band_wrap.

The rule: the input grid's rows are each a single solid color. Reading top to bottom, those row colors form a sequence of bands (runs of identical colors). The output is a square grid of concentric rectangular rings. The first band paints the outermost ring, the second band paints the next ring inward, and so on, until the last band fills the center square. Each ring has thickness equal to its band's row count. Output size: S = 2 * TotalBandCount - LastBandCount.

The core algorithm is a single formula: cell (R, C) in the output grid belongs to ring min(R, C, S-1-R, S-1-C). That is the distance to the nearest edge. Find which band bracket contains that distance, and you have the color. No iteration, no fill, no recursion over cells - one lookup per cell.

What makes this task elegant: the input IS the recipe. The sequence of row colors and their heights directly encodes the ring structure. The input is a compressed specification of the output.
