ABSTRACT

We report Rung 22 of Mentova's 48-rung reasoning ladder: diagrammatic reasoning.
Mentova reads a small grid layout and correctly interprets spatial patterns. For a
3×3 cross-shaped grid, Mentova correctly counts 5 occurrences of the symbol x,
identifies x as the centre cell, and characterises the grid as a cross pattern.
Pass criterion met: spatial pattern correctly interpreted.


INTRODUCTION

Diagrammatic reasoning is the ability to read, interpret, and reason about spatial
layouts, grids, and diagrams without reducing them to purely symbolic representations.
Rung 22 gives Mentova a grid reader that counts symbols, identifies positions, finds
patterns, and locates centre cells.


BACKGROUND

Diagrammatic reasoning has been studied in AI since the work of Larkin and Simon
(1987) on why a diagram is worth 10,000 words. Grid-based reasoning is used in
visual puzzles, matrix problems, and robot map reading.


METHOD

A new module, src/mentova/diagrammatic.pl, implements:

    mentova_diagrammatic(+Query, -Result, -Justification)

Supported queries: read_cell, count, find, dimensions, diagonal, centre, pattern.

Named grids stored: example (4×4), simple3x3, cross (3×3).


DEMONSTRATION

    % Query Mentova for diagrammatic reasoning about 'centre(cross)'.
    ?- mentova_query(diagrammatic, centre(cross), R).
    R = answer(x, just(diagrammatic(cross, centre_cell(x))))

    The centre cell of the 3×3 cross grid is x.

    % Query Mentova for diagrammatic reasoning about 'count(cross'.
    ?- mentova_query(diagrammatic, count(cross, x), R).
    R = answer(5, just(diagrammatic(cross, symbol(x), count(5))))

    Five x symbols in the cross grid.


RESULTS

Centre cell correctly identified as x. Symbol count correctly computed as 5.
Pass criterion met: spatial pattern correctly interpreted.


SIGNIFICANCE

Diagrammatic reasoning enables Mentova to work with grid-based puzzles, maps,
and visual representations, extending beyond purely symbolic knowledge.


HONEST LIMITATIONS

Only named grids stored in the knowledge base can be read. Dynamic grid input
and complex visual parsing are not yet implemented.


REPRODUCIBILITY

Requirements: SWI-Prolog 9.0.4 or later.

  git clone https://github.com/ai-university-aiu/Mentova.git
  cd Mentova
  % Load the Mentova bootstrap into SWI-Prolog, execute the query, and halt.
  swipl -l src/mentova/mentova.pl \
        % Supply a goal or termination instruction to the SWI-Prolog command.
        -g "mentova_boot, \
            % Query Mentova for diagrammatic reasoning about 'centre(cross)'.
            mentova_query(diagrammatic, centre(cross), R1), write(R1), nl, \
            % Query Mentova for diagrammatic reasoning about 'count(cross'.
            mentova_query(diagrammatic, count(cross,x), R2), write(R2), nl" \
        % Supply a goal or termination instruction to the SWI-Prolog command.
        -t halt
