Accomplishment 122: Tiling, Stamping, and Period Detection - The Tile Pack

Author: D. R. Dison, Founder of AIU, Creator of PrologAI and Mentova, ORCID 0009-0001-9246-5758, linkedin.com/in/d-r-dison/

Date: 2026-06-25

Abstract

Mentova's cognitive substrate gains its 100th work package: the tile pack (ti_* predicate family, Layer 81). This pack provides 14 predicates for composing grids from repeating motifs, splitting grids into sub-grids, stamping motifs at specific positions, extracting named tile positions, verifying that a grid is a tiling of one motif, detecting horizontal and vertical periodicity, and generating checkerboard patterns.

Introduction

Many visual reasoning tasks involve structure that repeats - a motif tiled across a canvas, a pattern with a known period, a stamp placed at multiple positions, or a grid that can be decomposed into uniform tiles. Before the tile pack, these operations required composing lower-level predicates in ad hoc ways for each task.

The tile pack introduces a coherent vocabulary for tiling operations. It answers questions like: what motif tiles this grid? how many times does this pattern repeat horizontally? where do I stamp this shape? split this grid into its constituent tiles.

Predicates

The tile pack exports 14 predicates under the ti_* prefix.

Tiling:
ti_tile_h/3      repeat a tile N times side-by-side horizontally
ti_tile_v/3      repeat a tile N times top-to-bottom vertically
ti_tile/4        tile a motif into NR rows of NC copies

Splitting:
ti_split_rows/3  split a grid into horizontal TH-row bands
ti_split_cols/3  split a grid into vertical TW-col stripes
ti_split/4       split into a list-of-tile-rows of TH x TW tiles
ti_flatten_tiles/2  reassemble tiles back into one grid

Stamping:
ti_stamp/5       overlay a motif at position (R, C)
ti_stamp_all/4   stamp a motif at every R-C position in a list

Extraction and verification:
ti_extract_tile/6  extract the TH x TW tile at tile-position (TR, TC)
ti_is_tiling/3     check if a grid is an exact tiling of one motif

Period detection:
ti_find_period_h/2  smallest horizontal period in number of columns
ti_find_period_v/2  smallest vertical period in number of rows

Pattern generation:
ti_checkerboard/5   generate an H x W checkerboard with two values

Implementation

ti_tile_h and ti_tile_v use foldl over a numlist to append N copies of
a row or the full tile grid. ti_tile composes them: ti_tile_h(Tile, NC)
followed by ti_tile_v(HGrid, NR).

ti_split_rows and ti_split_cols both use maplist with numlist-generated
index ranges to extract rows and columns respectively. ti_split composes
them: split into bands, then split each band into stripes. ti_flatten_tiles
inverts this: for each tile-row, extract each row index from every tile and
concatenate, then stack all bands with append.

ti_stamp uses maplist/4 over row and column indices, computing
motif-relative offsets (MRI = RI - R, MCI = CI - C). When an offset is
negative or out of bounds, the base cell is preserved. ti_stamp_all folds
ti_stamp over a list of R-C positions.

ti_is_tiling extracts all TH x TW tiles using ti_extract_tile and compares
each to the reference tile at (0,0) using structural equality (==). Forall
checks universally.

ti_find_period_h uses between/3 to search periods 1..NCols in ascending
order, committing with a cut to the first that makes NCols mod P = 0 and
every row have period P. ti_find_period_v does the same for rows.

ti_checkerboard assigns V1 or V2 by (RI + CI) mod 2.

Significance

The tile pack reaches work package 100 - a milestone in the platform's growth.

Tiling operations are fundamental to visual pattern tasks. A grid that is
a 2x2 tiling of a motif encodes structural repetition. A period of 3 in a
row means the row's pattern unit is 3 cells wide. Stamping a motif at
multiple positions builds composite patterns. With the tile pack, Mentova
can now:

  Detect: is this grid a tiling? what motif does it repeat?
  Compose: build large periodic grids from small motifs
  Decompose: split a grid into its constituent tiles
  Stamp: place shapes at computed positions
  Verify: does a given motif tile this grid exactly?

These capabilities directly support the class of ARC-AGI-2 tasks that
involve repeating patterns, periodic structure, and tiled compositions.
