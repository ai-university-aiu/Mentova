ABSTRACT

We report Rung 43 of Mentova's 48-rung reasoning ladder: intuitive reasoning.
Mentova uses prototype-based pattern matching to make rapid classifications.
Given features [has_wings, has_feathers, can_fly, lays_eggs], classification
is bird (robin exemplar) with confidence 1.0. Given [cold_blooded, has_scales],
gut reaction is fish (triggered by cold_blooded). Pass criterion met: rapid
classification without deliberation, with matching prototype named.


INTRODUCTION

Intuitive reasoning (Kahneman, 2011) is System 1 thinking: fast, associative,
prototype-based. Rung 43 implements prototype matching as a computational model
of intuition.


BACKGROUND

Prototype theory (Rosch, 1973) proposes that categories are defined by typical
members (prototypes) rather than necessary and sufficient conditions. Category
membership is graded by similarity to the prototype.


METHOD

A new module, src/mentova/intuitive.pl, implements:

    mentova_intuitive(+Query, -Result, -Justification)

Ten prototypes across three domains (animals, food, vehicles). Match scoring
by feature coverage. Gut reaction by first triggering feature.
Queries: classify, gut, prototypes, match_detail.


DEMONSTRATION

    % Query Mentova for intuitive reasoning about 'classify([has_wings'.
    ?- mentova_query(intuitive, classify([has_wings, has_feathers, can_fly, lays_eggs]), R).
    R = answer(match(bird, robin, confidence(1.0)), ...)

    % Query Mentova for intuitive reasoning about 'gut([cold_blooded'.
    ?- mentova_query(intuitive, gut([cold_blooded, has_scales]), R).
    R = answer(gut(fish, triggered_by(cold_blooded)), ...)


RESULTS

Full feature set: bird classified at confidence 1.0. Minimal cues: gut triggers fish.
Pass criterion met.


SIGNIFICANCE

Intuitive reasoning explains fast expert judgment, perceptual categorisation,
and heuristic decision-making. It complements deliberative reasoning.


HONEST LIMITATIONS

Prototype matching uses feature coverage, not typicality gradients. Real intuition
involves more complex similarity metrics and learned prototypes.


REPRODUCIBILITY

Requirements: SWI-Prolog 9.0.4 or later.

  git clone https://github.com/ai-university-aiu/Mentova.git
  cd Mentova
  % Load the Mentova bootstrap into SWI-Prolog, execute the query, and halt.
  swipl -l src/mentova/mentova.pl \
        % Supply a goal or termination instruction to the SWI-Prolog command.
        -g "mentova_boot, \
            % Query Mentova for intuitive reasoning about 'classify([has_wings'.
            mentova_query(intuitive, classify([has_wings,has_feathers,can_fly,lays_eggs]), R), \
            write(R), nl" \
        % Supply a goal or termination instruction to the SWI-Prolog command.
        -t halt
