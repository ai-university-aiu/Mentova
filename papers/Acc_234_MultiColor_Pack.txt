Abstract

PrologAI gains its 214th work package: the multicolor pack, a suite of 14 mc_*
predicates for querying and partitioning obj(Color,Cells) scenes by color
frequency and membership.


Motivation

Many ARC-AGI-2 tasks require reasoning about color distribution. "Which color
appears most often?" "Are there any colors that appear exactly once?" "Do any
two colors appear the same number of times?" The multicolor pack provides
clean, composable answers to all of these questions.


Color Frequency Queries

mc_color_counts returns all Color-N pairs sorted by color, giving a complete
frequency census of the scene. mc_dominant_color and mc_rarest_color find the
extremes. mc_color_freq_rank returns the Nth most frequent color. mc_color_index
returns the 1-based frequency rank of a specific color.


Membership and Presence

mc_color_present checks if a color exists. mc_count_of_color gives the exact
count (0 if absent). mc_n_distinct_colors counts how many colors are present.


Singleton and Majority Queries

mc_singleton_colors returns all colors with exactly one object. mc_unique_color
is a backtrackable version. mc_majority_color finds the color that holds more
than half of all objects (fails if none does).


Structural Queries

mc_color_partition groups objects by color: Color-Objs pairs. mc_pair_colors
returns all unordered color pairs. mc_equal_count_pairs finds color pairs that
have the same object count.


Test Coverage

42 of 42 unit tests pass. Tests cover all 14 predicates: counts, presence,
dominance, rarity, singleton, unique, freq rank, partition, pairs, equal-count
pairs, majority, distinct count, and color index.


Significance

The multicolor pack provides the color-statistics layer for the ARC-AGI-2
reasoning stack. Combined with scene, colortable, transformgen, and gridsolve,
it supports a wide class of visual transformation tasks that require
distributional reasoning over object colors.
