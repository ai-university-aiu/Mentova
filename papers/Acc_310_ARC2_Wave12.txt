ABSTRACT

Mentova solves ARC-AGI-2 task 36a08778 using a new glass-box named rule called
waterfall. The rule models 6-coloured seed cells as water sources that flow
downward by gravity through background cells, spreading horizontally around
solid obstacle (non-background, non-6) cells. The BFS simulation terminates
spreading at drain points (background cells reachable below) or at barriers
(existing 6-cells, obstacles, or grid boundaries). Out-of-bounds below does not
trigger spreading. The rule is verified on all 6 training pairs and the test
case of task 36a08778, raising the ARC-AGI-2 score from 11/120 = 9.17% to
12/120 = 10.00% - the first double-digit percentage milestone.

1. PROBLEM STATEMENT

Task 36a08778 presents grids where a small number of 6-valued seed cells appear
near the top. The rest of the grid contains background (value 7) and scattered
rows of non-background, non-6 cells (value 2, acting as solid obstacles). All
output changes are from background to 6; no other colours are altered. The
transformation has 6 training pairs ranging from 8x9 to 15x16 grids (28 to 65
changes each). A single consistent rule must explain all six training examples
and correctly predict the test output.

2. KEY INSIGHT - GRAVITY FLOW

The visual pattern in the outputs is unmistakable once identified: the 6-values
trace paths outward from the seeds, flowing downward and then bending around
the edges of obstacle rows exactly as water would around solid objects. The 6
fills never cross obstacle cells and never appear in open background regions
that are unreachable from any seed by a downward-then-sideways path.

The critical algorithmic insight is the distinction between two types of
blockage below a 6-cell:

- Out-of-bounds (OOB) below: the cell is at the bottom row of the grid. Water
  simply stops here; no spreading occurs.

- In-grid obstacle below: a non-background, non-6 cell blocks downward flow.
  Water spreads horizontally LEFT and RIGHT in the current row.

A second insight is the spread-stop condition: horizontal spreading ends when
the algorithm encounters either (a) a barrier cell (existing 6, obstacle, or
grid boundary) meaning water pools there, or (b) a drain point, a background
cell whose cell directly below is also background, meaning water can resume
falling from that position. The drain cell is INCLUDED in the spread range.

3. RULE SPECIFICATION (waterfall)

Inputs:
  - Grid: list of lists of integers
  - Bg: background colour (most frequent value; here 7)

Initialisation:
  - Seeds: all cells (R,C) where Grid[R][C] = 6
  - PQ: sorted priority queue (row-major order) seeded with all seed cells
  - Vis: set of all cells ever enqueued, initialised to the seed set

Main loop (BFS, row-major priority):
  Pop cell (R,C) from PQ.
  Let BV = effective value of cell (R+1, C):
    - If (R+1,C) is in Vis: BV = 6
    - Else if R+1 >= Nr: BV = OOB
    - Else: BV = Grid[R+1][C]

  Case 1 - BV == Bg (background): flow down.
    If (R+1,C) not in Vis: mark it, enqueue it, add to New.

  Case 2 - BV is in-grid obstacle (not Bg, not 6, not OOB): spread.
    Find left edge Lc and right edge Rc in row R:
      Spread left from C: move to C-1, C-2, ... until:
        (a) next cell is non-background (barrier) -> stop at current C
        (b) next cell is background AND its below is Bg (drain) -> include it, stop
        (c) grid boundary -> stop at current cell (pool)
      Spread right from C: symmetric.
    For each background cell in [Lc,Rc] not yet in Vis: mark it, enqueue it.

  Case 3 - BV is 6 or OOB: do nothing.

Termination: empty PQ.

Result: apply all marks (set marked cells to 6) in the original grid.

7. SIGNIFICANCE

The 10.00% milestone marks Mentova's first double-digit percentage on ARC-AGI-2.
The waterfall rule exemplifies PrologAI's glass-box philosophy: a named,
human-readable algorithm that can be stated precisely, inspected directly, and
proven correct on every training example before being applied to test cases.
Unlike opaque neural approaches, the rule can be audited at every step.

The gravity-flow pattern discovered here also generalises: many ARC tasks
involve physics-like propagation rules (gravity, spreading, flooding). The
waterfall predicate structure (priority-queue BFS with directional constraints)
provides a reusable template for future rules of this family.
