ABSTRACT

Mentova solves ARC-AGI-2 task 2ba387bc using a new glass-box named rule called
shape_sort.

The input grid contains several 4x4 colored blocks placed on a background of
zeros. Each block is either HOLLOW (a one-cell-wide border of one color
surrounding a 2x2 interior of zeros) or SOLID (all 16 cells the same color).
The rule separates the blocks into two typed lists, preserving reading order
(top-left row, then column) within each list. It then pairs the lists at equal
positions: hollow[i] with solid[i], side by side in 4 rows of 8 columns. When
one list is longer, the missing partner is replaced by a 4x4 block of zeros.
The pairs are stacked vertically to form the output grid.

All four training pairs and the held-out test case are solved correctly on
first attempt.


RULE DESCRIPTION

1. Find all distinct non-background (non-zero) colors in the input grid.

2. For each color, locate the top-left corner of its 4x4 bounding box by
   computing the minimum row and minimum column among all cells of that color.

3. Extract the 4x4 subgrid at that corner.

4. Sort all found shapes in reading order (ascending row, then ascending
   column of the top-left corner). This is the canonical order for both
   classification lists.

5. Classify each shape:
   - HOLLOW: the 4x4 subgrid contains at least one zero.
   - SOLID: the 4x4 subgrid contains no zeros.

6. Separate the sorted list into a HOLLOW list and a SOLID list, each
   retaining the original reading order.

7. Pair the lists: position i gives [hollow[i], solid[i]]. If one list is
   shorter, substitute a 4x4 block of zeros for the missing partner.

8. Render each pair as 4 rows of 8 columns (left block = hollow or zero,
   right block = solid or zero).

9. Stack all pair blocks vertically to form the output grid. Output height =
   4 * max(|hollow|, |solid|) rows; output width = 8 columns.


EXAMPLE

Training pair 1 (input: 23 rows x 24 cols, output: 16 rows x 8 cols):

Input shapes and their reading-order positions:
  1. color=1, top-left=(1,2),  SOLID
  2. color=2, top-left=(4,9),  SOLID
  3. color=8, top-left=(6,17), HOLLOW
  4. color=7, top-left=(12,1), SOLID
  5. color=3, top-left=(13,8), HOLLOW
  6. color=6, top-left=(15,17),SOLID
  7. color=4, top-left=(18,3), HOLLOW

Hollow list (reading order): [8, 3, 4]
Solid  list (reading order): [1, 2, 7, 6]

Pairs:
  Position 1: hollow=8 | solid=1  -> rows 0-3  of output
  Position 2: hollow=3 | solid=2  -> rows 4-7  of output
  Position 3: hollow=4 | solid=7  -> rows 8-11 of output
  Position 4: hollow=(zero) | solid=6 -> rows 12-15 of output

Output rows 0-3:
  [8,8,8,8,1,1,1,1]
  [8,0,0,8,1,1,1,1]
  [8,0,0,8,1,1,1,1]
  [8,8,8,8,1,1,1,1]

(remaining pairs follow the same pattern)
