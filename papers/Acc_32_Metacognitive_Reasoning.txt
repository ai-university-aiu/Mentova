ABSTRACT

We report Rung 32 of Mentova's 48-rung reasoning ladder: metacognitive reasoning.
Mentova reasons about its own capabilities, confidence levels, and what it can
and cannot do. Given a capability tag such as "causal," Mentova reports its
confidence level (high). Given "can_i_do(dialectical)" it answers yes with a
description. Mentova self-describes as having 28 high-confidence capabilities.
Pass criterion met: Mentova reports its own capability profile.


INTRODUCTION

Metacognition — thinking about thinking — is a hallmark of sophisticated
reasoning. Rung 32 gives Mentova a capability registry and introspective
predicates so it can answer questions about its own mind.


BACKGROUND

Metacognitive AI (Cox, 2005) enables systems to monitor their own reasoning,
allocate cognitive effort, and know the limits of their competence. This
implementation is a declarative capability registry rather than runtime monitoring.


METHOD

A new module, src/mentova/metacognitive.pl, implements:

    mentova_metacognitive(+Query, -Result, -Justification)

Queries: what_can_i_do, confidence_for(Tag), self_describe, can_i_do(Tag).


DEMONSTRATION

    ?- mentova_query(metacognitive, confidence_for(causal), R).
    R = answer(conf(causal, high), just(metacognitive(confidence_query(causal), confidence(high))))

    ?- mentova_query(metacognitive, self_describe, R).
    R = answer(desc(mentova, 28, prologai), ...)


RESULTS

Mentova correctly reports high confidence for all 28 implemented reasoning types.
Pass criterion met.


SIGNIFICANCE

Metacognitive reasoning enables trust calibration: a user can ask Mentova whether
it can handle a given reasoning type before submitting a query. It also enables
graceful degradation when queried outside its competence.


HONEST LIMITATIONS

The capability registry is static; it does not update based on runtime performance.
True metacognition would monitor reasoning quality dynamically.


REPRODUCIBILITY

Requirements: SWI-Prolog 9.0.4 or later.

  git clone https://github.com/ai-university-aiu/Mentova.git
  cd Mentova
  swipl -l src/mentova/mentova.pl \
        -g "mentova_boot, mentova_query(metacognitive, self_describe, R), write(R), nl" \
        -t halt
