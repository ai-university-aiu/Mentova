ABSTRACT

Mentova solves ARC-AGI-2 task 16de56c4 using a new glass-box named rule called
period_extend.

The input grid contains sparse non-background cells arranged in periodic patterns
along rows or columns. Each active line holds an "anchor pair" - two cells of the
same color separated by some step D. Extending the pattern to all positions sharing
the same residue modulo D fills the line. When a cell of a different color appears
at a pattern position, it acts as a stop-and-recolor marker: the fill uses the
marker's color and stops at the marker position rather than continuing to the grid
boundary. Non-background cells at non-pattern positions (singletons) are preserved
unchanged.

The rule determines automatically whether to operate row-wise or column-wise by
counting active lines in each dimension and using the direction with more.

All three training pairs and the held-out test case are solved correctly on first
attempt.


RULE DESCRIPTION

An active line is any row or column that contains two or more cells of the same
non-background (0) color. For each active line:

1. Find the anchor pair: the two closest cells of the same color. Their separation
   is the step D.

2. Compute all pattern positions: every index P in the line such that
   P mod D equals the anchor positions modulo D.

3. If a cell at a pattern position holds a different non-zero color, it is
   the stop/recolor cell. Fill all pattern positions from the anchor side
   to the stop (inclusive) with the stop color; leave positions beyond the
   stop as background.

4. If no stop cell exists, fill every in-bounds pattern position with the
   anchor color.

5. Non-background cells at non-pattern positions (singletons) are copied
   to the output unchanged.

Direction (row vs column) is determined by counting how many rows have same-color
pairs versus how many columns have same-color pairs. The dimension with more active
lines wins.


EXAMPLE

Training pair 1 (row mode, 12 x 9):

Row 8 in the input: [9, 0, 9, 0, 0, 0, 6, 0, 0]
- Anchor: 9s at columns 0 and 2, step D = 2
- Pattern positions: 0, 2, 4, 6, 8
- Stop/recolor cell: 6 at column 6 (a pattern position, different color from 9)
- Fill range: columns 0 to 6 (anchor is at left, stop at right)
- Fill color: 6 (the stop color)
- Output row 8: [6, 0, 6, 0, 6, 0, 6, 0, 0]

Row 11 in the input: [8, 8, 0, 0, 0, 0, 0, 3, 0]
- Anchor: 8s at columns 0 and 1, step D = 1
- Pattern positions: 0, 1, 2, ..., 8 (every column)
- Stop/recolor cell: 3 at column 7 (a pattern position, different color from 8)
- Fill range: columns 0 to 7 (anchor at left, stop at right)
- Fill color: 3
- Output row 11: [3, 3, 3, 3, 3, 3, 3, 3, 0]
