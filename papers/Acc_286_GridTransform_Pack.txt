Abstract

PrologAI gains transformation detection between grids.

The gridtransform pack (Work Package 266, Layer 245, prefix gtr_*) provides 14
predicates for detecting color mappings between two grids, computing cell-level
differences, and applying transformations. Color maps are cm(Old, New) lists.
Change lists are chg(R, C, OldColor, NewColor) terms.

Background

ARC-AGI-2 tasks require a system to identify the rule that maps input grids to
output grids. The most fundamental class of such rules is color permutation: every
cell that was color A becomes color B, every B becomes C, and so on.

Beyond color permutation, the system needs to identify which cells changed, by how
much, and to reproduce that change on a new input. The gridtransform pack provides
all the primitives needed for this transformation pipeline.

The Pack

Module: gridtransform. File: packs/gridtransform/prolog/gridtransform.pl.
Prefix: gtr_. Layer: 245. Work Package: 266. PrologAI PR: 356.

Predicates:

gtr_color_map(+GridA, +GridB, -Map)
Detect the consistent color mapping from GridA to GridB as a sorted list of
cm(OldColor, NewColor) pairs. Fails if any source color maps to more than one
destination at different positions.

gtr_apply_color_map(+Grid, +Map, -Result)
Apply a cm/2 color map to a grid. Unmapped colors pass through unchanged.

gtr_diff_cells(+GridA, +GridB, -Cells)
List of r(R,C) positions where the grids have different values.

gtr_diff_count(+GridA, +GridB, -N)
Count of positions where the grids differ.

gtr_same_cells(+GridA, +GridB, -Cells)
List of r(R,C) positions where the grids agree.

gtr_changed_colors(+GridA, +GridB, -Changes)
List of chg(R,C,OldColor,NewColor) for each position that changed.

gtr_invert_map(+Map, -InvMap)
Invert a color map. Fails if the map is not injective.

gtr_compose_maps(+MapAtoB, +MapBtoC, -MapAtoC)
Compose two color maps. Only colors present in both maps appear in the result.

gtr_is_identity_map(+Map)
Succeed if every cm(VA,VB) in Map has VA = VB. Succeeds for empty map.

gtr_apply_changes(+Grid, +Changes, -Result)
Apply a list of chg/4 terms to a grid.

gtr_delta_grid(+GridA, +GridB, +Bg, -Delta)
Grid showing only changed cells: changed positions use GridB's value; unchanged
positions use Bg.

gtr_overlay(+BaseGrid, +DeltaGrid, +Bg, -Result)
Paste non-Bg cells of DeltaGrid onto BaseGrid.

gtr_common_grid(+GridA, +GridB, +Bg, -Common)
Keep cells where both grids agree; differing cells become Bg.

gtr_is_color_permutation(+GridA, +GridB)
Succeed if GridB is a bijective color permutation of GridA: consistent and injective.

Key Composition

The delta and overlay predicates compose cleanly:
  overlay(A, delta(A, B, Bg), Bg) = B

This means: given grid A and grid B, compute what changed (delta), then applying
those changes to A via overlay reconstructs B exactly.

The color map pipeline composes equally cleanly:
  Map = color_map(A, B)
  apply_color_map(A, Map) = B
  apply_color_map(B, invert_map(Map)) = A
