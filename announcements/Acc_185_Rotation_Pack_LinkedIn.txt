PrologAI gains its 165th work package: the rotation pack (ro_*, Layer 144).

14 predicates for quarter-turn reasoning on 2D grids.

ro_rot90 rotates a grid 90 degrees clockwise: each new row C is column C of
the original read bottom-to-top. For a 2x2 grid [[a,b],[c,d]], the result
is [[c,a],[d,b]]. Dimensions swap: H rows by W cols becomes W rows by H cols.

ro_rot180 uses maplist(reverse) then reverse. For [[a,b],[c,d]]: [[d,c],[b,a]].

ro_rot270 is 90 counter-clockwise: each new row C reads column W-1-C top-to-bottom.
For [[a,b],[c,d]]: [[b,d],[a,c]].

ro_rot_n dispatches by index N in 0..3 with cuts to prevent choicepoints.

ro_all produces all four rotations. ro_canonical returns the lexicographically
smallest, giving a unique representative for each rotation equivalence class.

ro_is_rot2 tests 2-fold symmetry (invariant under 180).
ro_is_rot4 tests 4-fold symmetry (invariant under 90, square grids only).
ro_sym_order returns 1, 2, or 4.

ro_rotate_cells rotates a list of R-C coordinate pairs. For N=3 (270 CW):
(R,C) -> (W-1-C, R). The new column is the old ROW, not the old column.

ro_spin2 overlays a grid with its 180 rotation; non-background cells win.
ro_spin4 overlays all four rotations to complete 4-fold symmetric patterns.

A single corner cell in a 2x2 grid:
ro_spin4([[1,0],[0,0]], 0, Out). Out = [[1,1],[1,1]].

ro_match_rotation finds which N in 0..3 maps GridA to GridB.
ro_equiv_rotation tests if any rotation of GridA equals GridB.

Round-trip verified: four 90-degree rotations return the original.
Three 90-degree rotations equal one 270-degree rotation.
