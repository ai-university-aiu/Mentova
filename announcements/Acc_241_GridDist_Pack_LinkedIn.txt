PrologAI now has a griddist pack - 14 gd_* predicates for computing spatial
distances between grid cells and for operations built on those distances.

What it does:

- gd_manhattan / gd_chebyshev: point-to-point L1 and Chebyshev distances
- gd_dist_map / gd_dist_map8: per-cell grids of min Manhattan or Chebyshev distance to nearest Color
- gd_nearest_coord: coordinates of the nearest Color cell from a given position
- gd_zone: mark cells within Manhattan distance N of any Color cell with atom zone
- gd_ring: all in-bounds cell positions at exactly Manhattan distance D from a center
- gd_voronoi: assign each background cell the color of its nearest foreground cell
- gd_bfs_flood: BFS distances from a start cell, treating Wall-color cells as impassable
- gd_recolor_by_dist: recolor each Color cell based on its distance from background (palette-driven)
- gd_expand_n: N-step morphological dilation — expand Color outward by N steps
- gd_shrink_n: N-step morphological erosion — shrink Color inward by N steps
- gd_equidistant: mark cells equidistant from ColorA and ColorB with atom equidist
- gd_max_dist: row and column of the cell farthest from any Color cell

53 of 53 unit tests pass.
