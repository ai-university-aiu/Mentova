Accomplishment 225: Scene-Level Uniform Transformation of All Objects (scenexf Pack, sx_*, Layer 184)

ORCID 0009-0001-9246-5758, linkedin.com/in/d-r-dison/

PrologAI now has a scenexf pack - 14 sx_* predicates that apply transformations
uniformly to every object in a scene (a list of obj(Color, Cells) terms).

Where the objxf pack transforms a single object, scenexf scales the same
class of operations to the entire list in one call. A symbolic reasoner can
now say: "recolor all red objects to blue", "shift the whole scene two rows
down", "keep only the largest object", or "remove all objects of background
color" - and get back the fully transformed scene instantly.

The 14 predicates cover color transformations (recolor, recolor_all,
apply_color_map), spatial transformations (shift, to_origin, reflect_h,
reflect_v), filtering (remove_color, keep_color), ordering (sort_size_desc,
sort_size_asc, sort_pos), and selection with deduplication (top_n, dedup_form).

Color swapping via a map list is fully supported. The deduplication predicate
uses normalized forms (translate-to-origin, sort cells) so it works correctly
regardless of where in the scene each object sits.

41 of 41 unit tests pass.
