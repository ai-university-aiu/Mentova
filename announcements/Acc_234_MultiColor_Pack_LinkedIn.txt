PrologAI now has a multicolor pack - 14 mc_* predicates for querying and
partitioning scenes by color frequency.

Core queries:
- mc_dominant_color: which color has the most objects?
- mc_rarest_color: which color has the fewest?
- mc_singleton_colors: which colors appear exactly once?
- mc_majority_color: is there a color that holds more than half the scene?
- mc_color_freq_rank: what is the Nth most frequent color?
- mc_color_index: what rank is a specific color by frequency?

Structural queries:
- mc_color_partition: group all objects by color
- mc_pair_colors: all unordered color pairs
- mc_equal_count_pairs: color pairs with the same object count

Plus: mc_color_counts, mc_count_of_color, mc_color_present, mc_n_distinct_colors.
