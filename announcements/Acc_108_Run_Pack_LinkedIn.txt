Accomplishment 108: Mentova gains run-length encoding for grid sequences - compressing row and column patterns into Value-Count pairs for structural analysis.

The run pack (rn_* predicates, Layer 67) gives Mentova 14 new tools for working with repeated patterns in sequences:

- rn_encode/2, rn_decode/2: encode [1,1,2,2,2,1] to [1-2,2-3,1-1] and back
- rn_row_encode/3, rn_col_encode/3: encode a single grid row or column
- rn_grid_rows/2, rn_grid_cols/2: encode all rows or all columns
- rn_length/2: total element count of a run list
- rn_at/3: value at a 0-indexed position in the encoded sequence
- rn_max_run/3: longest run of a given color (0 if absent)
- rn_count_runs/3: number of distinct runs of a given color
- rn_uniform/1: true when all runs share the same value
- rn_trim/3: remove leading and trailing background values from a list
- rn_repeat/3: repeat a sequence N times, merging boundary runs
- rn_positions/3: 0-indexed positions of all cells with a given value

With run-length encoding, Mentova can now describe stripes, measure band widths, test uniformity, and find where specific colors appear - all from a compact sequence representation that makes repeating structure explicit.
