The contrast pack answers the core question in learning from examples: which input features actually cause the output to change?

Given training pairs of input and output grids, the fourteen ca_* predicates identify features that co-vary with output changes - present in all changing pairs, absent from stable ones. This is the glass-box causal analysis that replaces learned weights with principled symbolic reasoning.

Key predicates: ca_pairwise_delta (what changed per pair), ca_covarying_features (causal candidates), ca_rank_features (sorted by correlation), ca_context_gate (the top trigger), ca_separates (precision scoring).

The contrast pack completes the four-pack learning stack: gridobjmatch detects object-level changes, symtab learns symbol semantics, invariant extracts what never changes, and contrast isolates what drives the changes.
