PrologAI now has a ruleinfer pack - 14 ri_* predicates that infer which
scene-level transformation was applied to a Before scene to produce an After
scene.

This pack closes the loop between scene comparison and scene transformation.
scenecmp tells you THAT something changed. scenexf applies the change. ruleinfer
tells you WHAT the rule was.

Given a Before-After pair, ruleinfer can determine:
- Was it a recolor? (ri_infer_recolor)
- Was every object set to the same color? (ri_infer_recolor_all)
- Was a color filter applied? (ri_infer_keep_color, ri_infer_remove_color)
- Was the scene shifted? (ri_infer_shift, ri_infer_to_origin)
- What was the full color substitution map? (ri_infer_color_map)

Once a candidate rule is found from one pair, consistency predicates verify it
works across all training pairs. Cross-pair queries check whether object counts
and color sets are preserved.
