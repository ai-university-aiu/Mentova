The twelve chapters progress from the absolute beginning - what a fact is, what a rule is, what a query is - all the way to the most advanced capabilities on the platform: the actor framework, episodic memory (SONA), the Piagetian and CHC assessment frameworks, and all four agent communication protocols (Model Context Protocol/MCP, Agent-to-Agent/A2A, Agent Communication Protocol/ACP, Agent Network Protocol/ANP).

The tutorial is available in two formats:

• Multi-file: docs/tutorial/ in the PrologAI repository. Twelve chapter files. Read one chapter at a time.

• Monolithic: docs/PrologAI_Tutorial.txt. All twelve chapters in one file. Download and convert to PDF.

The console REPL launcher (Acc_68, bin/mentova) is the interactive substrate for all tutorial examples. Every example in the tutorial runs at the ?- prompt.
