Accomplishment 100: Mentova now induces transformation rules from grid pairs.

The induction pack (id_* predicates, Layer 59) gives Mentova 14 new reasoning tools for analyzing what happened between an input grid and its output:

- Was this a color substitution? id_is_recolor/2 checks for consistent color mapping across every cell.
- What colors were added or removed? id_new_colors/3 and id_lost_colors/3 compute the color set delta.
- Which cells changed? id_changed_cells/3 returns the exact list.
- Did the grid scale up? id_is_scale/2 detects uniform integer scaling; id_scale_factor/3 returns the factor.
- What is the full color substitution map? id_color_map/3 infers the sorted Old-New pairs.

These are the observational primitives that let a system form a hypothesis from examples, rather than just measuring what it sees.
