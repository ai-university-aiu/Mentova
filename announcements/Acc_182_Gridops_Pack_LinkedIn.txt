14 predicates now reason across collections of multiple grids simultaneously.

go_always and go_never find positions that have a given value in all grids
or no grids.

go_sometimes finds positions that have a value in some grids but not all.

go_count_v builds a count grid: each cell holds how many source grids have
a specific value there.

go_modal computes the most frequent value per cell across all grids.
Ties broken by smallest value.

go_stable finds cells where all grids agree on the same value. go_unstable
finds cells where they disagree.

go_eq tests two grids for cell-for-cell identity via structural unification.

go_add, go_sub, go_emax, go_emin perform elementwise arithmetic across
two same-size grids.

go_overlay overlays multiple grids, taking the first non-background value
per cell.

go_intersect intersects multiple grids, keeping only cells where all grids
unanimously hold the same non-background value.
