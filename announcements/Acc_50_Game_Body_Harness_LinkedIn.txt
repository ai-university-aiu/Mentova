The game-as-a-body harness is live.

The design principle from Volume 6: build it once, point it at several targets.
We built it. We pointed it at four.

WHAT THE HARNESS DOES

Any game environment enrolls as a body through three predicates:
game_enroll, game_observe, game_act.

Game frames arrive as visual perception signals. Mentova reasons.
Actions go back as actuator commands. Every answer comes with a
human-readable justification chain.

THE RESULTS

ARC-AGI (3 tasks, all pass):
Mentova induces the transformation rule from training examples, applies
it to the test input, and proposes the correct output grid.

Task 1 - recolor: induced rule recolor_nonblack(2), verified on 3 pairs.
Output: [[2,0,0],[2,0,0],[2,2,0]]. Correct.

Task 2 - horizontal flip: induced rule horizontal_flip, verified on 3 pairs.
Output: [[0,2,1],[3,0,0],[0,0,4]]. Correct.

Task 3 - fill border: induced rule fill_border(5), verified on 3 pairs.
Border cells become grey. Interior cell preserved. Correct.

No pretraining. No neural network. Genuine induction from examples.
The justification names the induced rule, the number of training pairs verified,
and the method used.

Raven's Progressive Matrices (2 tasks, both correct):
Mentova induces the relational pattern from the 8 known matrix cells,
predicts the missing 9th cell, and selects the matching option.

Size progression matrix: induced size_increases_across_row. Predicted large
grey circle. Selected option 1. Correct.

Count progression matrix: induced count_increases_across_row. Predicted 3
grey squares. Selected option 3. Correct.

Baba Is You (2 levels, both planned):
Mentova reads the active game rules as node_facts, reasons about who the
player is and what the win condition is, and produces a plan.

Level 1 (BABA IS YOU, FLAG IS WIN): direct path to flag. 5-step plan.
Level 2 (ROCK IS YOU, FLAG IS WIN): adapts the plan to control rock.

The rules are node_facts. The planning is genuine rule-driven reasoning.

Pokemon (stub — needs emulator bridge):
The perceive-reason-act cycle runs on a documented stub. Full implementation
requires mGBA with a Lua socket bridge and the Pokemon Red RAM map.
The stub documents every step of the path.

WHY THIS MATTERS

Every other system that scores on ARC-AGI is a black box. You see the score.
You do not see the reasoning.

Mentova's ARC score on these tasks is 3/3. The justification for each answer
names the induced rule, the training pairs it was verified against, and the
transformation applied to the test input.

That is the differentiator. Not the score. The chain.
