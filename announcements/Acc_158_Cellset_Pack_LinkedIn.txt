Grid conversion:

cs_to_grid(Cells, H, W, Color, Bg, Grid)  % paint cell set onto canvas

Set algebra:

cs_union(A, B, Union)             % sorted set union

cs_subtract(A, B, Diff)           % set difference A minus B

Position operations:

cs_bbox(Cells, R0, C0, R1, C1)           % bounding box; fails if empty
cs_normalize(Cells, Norm, DR, DC)         % translate top-left to (0,0)

Queries:

cs_adjacent_bg(Cells, Adj)  % sorted 4-connected background neighbors

Shape and geometry:

cs_same_shape(A, B)               % shape equality after normalization
cs_rotate_90(Cells, H, Rotated)   % 90-degree CW rotation (H = grid height)
cs_mirror_h(Cells, W, Mirrored)   % horizontal mirror (W = grid width)

Key design insight: representing objects as sorted R-C lists enables
set operations that are impossible or awkward on dense grids.
cs_same_shape normalizes both sets to the origin and compares structurally.
cs_adjacent_bg gives the exact background ring surrounding any object.
cs_rotate_90 applies the formula new_R = C, new_C = H-1-R via maplist.
