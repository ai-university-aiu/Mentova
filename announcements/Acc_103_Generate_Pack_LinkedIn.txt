Accomplishment 103: Mentova can now generate grids from visual pattern rules.

The generate pack (ge_* predicates, Layer 62) gives Mentova 14 new construction tools:

- ge_uniform/4: fill with a single color
- ge_gradient_h/4, ge_gradient_v/4: horizontal and vertical gradients
- ge_checkerboard/5: alternating two colors
- ge_stripes_h/4, ge_stripes_v/4: horizontal and vertical stripe patterns
- ge_border_rect/5, ge_frame/5: bordered rectangle and frame ring
- ge_diagonal/4, ge_antidiagonal/4: diagonal patterns
- ge_identity_grid/3: N x N identity matrix pattern
- ge_cross/5: cross through the center row and column
- ge_from_map/3: build from a list of cell-color pairs
- ge_repeat_pattern/4: tile a smaller pattern to fill a larger grid

These tools close the hypothesis-verification loop. Mentova can now induce a rule from training examples, generate what the output should look like for the test input, and compare it against the actual output.
